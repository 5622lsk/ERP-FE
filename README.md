## Installation 🛠️

### 1. Clone the Repository

```bash
git clone https://github.com/ERP-proj/ERP-FE.git
```

### 2. Install Dependencies

```bash
cd ERP-FE
npm install
```

### 3. Run the Local Server

```bash
npm run dev
```
